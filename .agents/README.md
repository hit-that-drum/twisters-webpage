# Agent Technical Docs

This directory stores implementation notes for the JWT-only password reset flow.

- Human-readable docs (MkDocs + Mermaid): `.agents/docs/reset-password-flow.md`
- Machine-readable spec: `.agents/spec/reset-password-flow.json`
- MkDocs config: `.agents/mkdocs.yml`

## How to Preview

From repository root:

```bash
mkdocs serve -f .agents/mkdocs.yml
```
